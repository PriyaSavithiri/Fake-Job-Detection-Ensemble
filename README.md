# Fake Job Detection Ensemble

This project focuses on detecting fake job postings using NLP and machine learning. It compares different models, including traditional ML models, LSTM-based deep learning, and ensemble approaches.

The goal is to build a system that can classify job postings as real or fake based on text and related job information.

## What This Project Includes

- Exploratory data analysis
- NLP preprocessing
- Traditional machine learning models
- LSTM model
- Ensemble model experiments
- Model interpretability
- Simple app interface

## How to Run
- Clone the repository:
git clone https://github.com/YOUR-USERNAME/Fake-Job-Detection-Ensemble.git
- Open the project folder:
cd Fake-Job-Detection-Ensemble
- Open the Jupyter Notebook files.
- Run the notebooks cell by cell in order.

There is no separate requirements file included. Install any missing Python libraries manually when the notebook asks for them.
